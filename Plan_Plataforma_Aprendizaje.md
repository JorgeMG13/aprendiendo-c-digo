# Especificación del proyecto

## Objetivo

Crear una plataforma web para aprender programación completamente en el
navegador, sin backend. Todo el progreso se guarda en LocalStorage.

## Tecnologías

-   HTML
-   CSS
-   JavaScript (ES Modules)
-   LocalStorage
-   JSON para contenidos

## Funciones principales

-   Cursos de HTML, CSS, JavaScript, Python, C++, Java, Git.
-   Mini documentación tipo referencia rápida.
-   Ejemplos interactivos.
-   Editor para HTML/CSS/JS.
-   Ejercicios.
-   Corrección automática.
-   Puntuación (0-100).
-   Feedback con errores y mejoras.
-   Proyectos finales guiados.
-   Modo "Aprender creando".
-   Ruta de videojuegos (Pygame, SFML, SDL2, LibGDX, Phaser, Unity
    introductorio).

## Funciones extra

-   Repetición inteligente.
-   Logros.
-   Estadísticas.
-   Diccionario.
-   Playground.
-   Encuentra el error.
-   Modo examen.
-   Ruta de aprendizaje.
-   Personalización.
-   Favoritos.
-   Buscador global.
-   Exportar e importar progreso (JSON).
-   Modo oscuro.

## Estructura recomendada

-   index.html
-   css/
-   js/
-   data/
-   assets/

## Guardado

Usar LocalStorage para: - progreso - favoritos - ajustes - logros -
estadísticas - proyectos

## Evaluación

-   Comparar resultado esperado.
-   Detectar errores frecuentes.
-   Dar pistas antes de mostrar solución.
-   Puntuar claridad y buenas prácticas cuando sea posible.

## Riesgos

-   LocalStorage limitado (\~5-10 MB según navegador).
-   No ejecutar Python/C++/Java localmente; usar preguntas, completar
    código o proyectos guiados.
-   Evitar perder datos: exportación manual e importación.
-   No depender de internet.
-   Mantener estructura modular.

## Diseño

-   Responsive.
-   Accesible.
-   Navegación sencilla.
-   Carga rápida.

## Fases

1.  Base de la web.
2.  Navegación.
3.  Cursos.
4.  Ejercicios.
5.  Corrección.
6.  Proyectos.
7.  Estadísticas.
8.  Exportación.
9.  Pulido y pruebas.
